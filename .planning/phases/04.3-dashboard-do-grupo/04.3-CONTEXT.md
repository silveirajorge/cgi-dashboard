## Phase Boundary

Dashboard de KPIs do grupo de avaliação de desempenho: cards com médias, gráfico comparativo entre funcionários, filtros por período e funcionário. Página em `/dashboard/avaliacao-dashboard`.

Decisões da Fase 4 (contexto geral) aplicam-se — ver D-19 a D-24 em `04-CONTEXT.md`.

</domain>

<decisions>
## Implementation Decisions

### Sidebar
- **D4.3-01:** Item "Dashboard" no grupo "Avaliação e Desempenho" passa de `comingSoon: true` para `comingSoon: false`, URL `/dashboard/avaliacao-dashboard`

### Página
- **D4.3-02:** Server Component page em `src/app/(main)/dashboard/avaliacao-dashboard/page.tsx`
- **D4.3-03:** Client components co-locados em `_components/`
- **D4.3-04:** Segue padrão do KPI dashboard de pedidos: filtros no topo, cards, gráficos abaixo

### KPIs (Cards)
- **D4.3-05:** Média geral do grupo (período selecionado)
- **D4.3-06:** Média por categoria (6 cards ou tabela com as médias de cada categoria)
- **D4.3-07:** Total de avaliações no período
- **D4.3-08:** Funcionário com melhor média

### Gráfico Comparativo
- **D4.3-09:** Gráfico de barras (BarChart Recharts) — eixo X = funcionários, eixo Y = média geral
- **D4.3-10:** Cada barra representa um funcionário, cor por funcionário
- **D4.3-11:** Ao clicar numa barra, navegar para o histórico do funcionário (Fase 4.4)

### Filtros
- **D4.3-12:** Dropdown de funcionário (ou "Todos") — apenas funcionários ativos
- **D4.3-13:** Filtro de período (date range) — mesmo padrão do dashboard de pedidos
- **D4.3-14:** Apenas funcionários ativos entram nas métricas

### API
- **D4.3-15:** `GET /api/avaliacoes/stats?from=&to=&funcionario_id=` — retorna:
  - `media_geral` — média de todas as avaliações no período
  - `media_categorias` — `{ pontualidade, qualidade, ... }` médias por categoria
  - `total_avaliacoes` — contagem
  - `melhor_funcionario` — `{ id, nome, media }`
  - `comparativo` — `[{ funcionario_id, funcionario_nome, media }]` para o gráfico de barras

### the Agent's Discretion
- Cores específicas das categorias nos cards
- Layout exato dos cards (grid cols responsivo)
- Ícone do sidebar (já definido como LineChart, mas pode mudar)

</decisions>
