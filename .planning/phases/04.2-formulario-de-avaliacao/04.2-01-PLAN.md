<objective>
Implementar a página de formulário de avaliação individual com 6 categorias (escala 1-10), média automática via SQLite GENERATED column, comentário opcional, listagem com filtros e detalhes apenas-leitura em modal.

**Purpose:** Permitir que o usuário registre avaliações de desempenho individuais para cada funcionário, com notas por categoria, média automática e comentário opcional — dados armazenados no SQLite para uso nas fases seguintes (Dashboard do Grupo, Histórico por Funcionário).

**User story:** **Como um** administrador, **eu quero** registrar avaliações de desempenho individuais com notas por categoria e média automática, **para que** possa acompanhar e analisar o desempenho de cada funcionário ao longo do tempo.

**Output:**
- Migração SQLite da tabela `avaliacoes` com GENERATED column para média
- API endpoints: GET /api/avaliacoes (listar com filtros), POST /api/avaliacoes (criar), GET /api/avaliacoes/[id] (detalhes)
- Página `/dashboard/avaliacao` com modo listagem (tabela + filtros) e modo formulário inline
- Modal de detalhes apenas-leitura com todas as notas por categoria
- Sidebar item "Avaliação" já existe sem `comingSoon` — nenhuma modificação necessária
</objective>

      // Validações

      if (!dataAvaliacao) { setError("Selecione uma data"); return; }
      for (const [key, value] of Object.entries(notas)) {

      try {
        const res = await fetch("/api/avaliacoes", {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({

            data_avaliacao: dataAvaliacao,
            ...notas,

          }),

      } finally {
