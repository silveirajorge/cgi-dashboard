Purpose: Permitir que usuários enviem CSVs diretamente pela interface do dashboard, sem linha de comando.
Output: Página de upload funcional com item na sidebar.
</objective>

  <action>
    Criar `src/app/(main)/dashboard/upload/_components/upload-area.tsx` como Client Component com `"use client"`.

    **Estados gerenciados com useState:**
    - `isDragOver: boolean` — true quando usuário arrasta arquivo sobre a área
    - `isUploading: boolean` — true durante o fetch para o servidor

    **Refs:**
    - `inputRef: useRef<HTMLInputElement>(null)` — referência para o `<input type="file" hidden>`

    **Função handleFile(file: File):**
    1. Valida extensão: `!file.name.toLowerCase().endsWith(".csv")` → `toast.error("Apenas arquivos CSV são aceitos")` + return
    2. Valida tamanho: `file.size > 50 * 1024 * 1024` → `toast.error("Arquivo excede o limite de 50MB")` + return
    3. `setIsUploading(true)`
    4. Cria `FormData`, faz `formData.append("file", file)` — nome do campo EXATAMENTE "file"
    5. `fetch("/api/upload", { method: "POST", body: formData })`
    6. Se `!res.ok`: throw com `data.error || "Erro ao processar upload"`
    7. Se sucesso: `toast.success(\`Upload realizado: ${data.rows_imported} linhas importadas, ${data.rows_ignored} ignoradas\`)`
    8. catch: `toast.error(message)` — exibe mensagem do servidor
    9. finally: `setIsUploading(false)`, reseta input: `if (inputRef.current) inputRef.current.value = ""`

    **Drag handlers:**
    - `handleDragOver(e)`: `e.preventDefault()`, `setIsDragOver(true)`
    - `handleDragLeave(e)`: `e.preventDefault()`, `setIsDragOver(false)`
    - `handleDrop(e)`: `e.preventDefault()`, `setIsDragOver(false)`, `handleFile(e.dataTransfer.files[0])`

    **Click handler:**
    - `handleClick()`: se `!isUploading`, `inputRef.current?.click()`
    - `onKeyDown`: Enter ou Space dispara `handleClick()` (acessibilidade)

    **Input change:**
    - `handleInputChange(e)`: se `e.target.files?.length`, `handleFile(e.target.files[0])`

    **Render — estrutura exata:**

    ```tsx
    return (
      <div className="flex flex-col gap-4">
        <h1 className="text-2xl font-semibold">Upload de CSV</h1>

        <div
          role="button"
          tabIndex={0}
          onDragOver={handleDragOver}
          onDragLeave={handleDragLeave}
          onDrop={handleDrop}
          onClick={handleClick}
          onKeyDown={(e) => {
            if (e.key === "Enter" || e.key === " ") handleClick();
          }}
          className={cn(
            "flex cursor-pointer flex-col items-center justify-center gap-2",
            "rounded-lg border-2 border-dashed p-12",
            "transition-colors duration-200",
            isUploading && "pointer-events-none opacity-70",
            isDragOver
              ? "border-primary bg-accent"
              : "border-border hover:border-primary hover:bg-accent/50",
          )}
        >
          <input
            ref={inputRef}
            type="file"
            accept=".csv"
            className="hidden"
            onChange={handleInputChange}
          />

          {isUploading ? (
            <Loader2 className="size-12 animate-spin text-muted-foreground" />
          ) : (
            <Upload
              className={cn(
                "size-12",
                isDragOver ? "text-primary-foreground" : "text-muted-foreground",
              )}
            />
          )}

          <p className="text-base font-medium">
            {isUploading
              ? "Enviando..."
              : isDragOver
                ? "Solte o arquivo aqui"
                : "Arraste o arquivo CSV aqui"}
          </p>

          {!isUploading && (
            <p className="text-xs text-muted-foreground">
              ou clique para selecionar &middot; CSV até 50MB
            </p>
          )}
        </div>
      </div>
    );
    ```

    **Imports obrigatórios:**
    ```tsx
    "use client";

    import { useState, useRef, type DragEvent, type ChangeEvent } from "react";
    import { Upload, Loader2 } from "lucide-react";
    import { toast } from "sonner";
    import { cn } from "@/lib/utils";
    ```

    **Regras:**
    - NÃO importar `useEffect`, `useCallback`, `useMemo` — não são necessários
    - NÃO usar react-hook-form ou qualquer biblioteca de formulário (D-05: upload automático sem botão)
    - NÃO usar react-dropzone (RESEARCH: HTML5 nativo é suficiente)
    - NÃO usar estado global (Zustand) — estado local com useState
    - Toast via `toast.success()` e `toast.error()` da sonner (já configurada no root layout)
    - Ícones `Upload` e `Loader2` da lucide-react (já instalada)
    - Per D-06 e D-07: validação client-side ANTES de enviar
    - Per D-13: sem redirecionamento após upload — apenas toast
    - Per D-14: validações client-side (extensão .csv + tamanho 50MB) antes de enviar
  </action>
